A beginner-friendly Artificial Intelligence project developed using Python that simulates human conversation through predefined decision rules.

This chatbot processes user input and generates responses using conditional logic instead of machine learning models.

# 📖 Project Overview

Chatbots are one of the most common applications of Artificial Intelligence.

This project demonstrates how conversational systems can be built using simple rule-based techniques before moving into advanced NLP and Machine Learning.

The chatbot receives text input, checks predefined conditions, and returns suitable responses.

---

# 🎯 Problem Statement

Create a chatbot that:

- Accepts user input
- Responds to greetings
- Handles predefined conversation patterns
- Ends interaction through exit commands
- Maintains continuous interaction

---

# 🚀 Objectives

The objective of this project is to understand:

- Human–computer interaction
- Decision-making logic
- Control flow implementation
- Basic chatbot architecture
